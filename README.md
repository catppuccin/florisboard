## USAGE

1. Download the file
2. Open the app
3. Click on Theme
4. Click on `Manage Installed Themes`
5. Import
6. Click on `Select files`
7. Select the file in the file manager
8. It will display the theme info and click on `Import`
9. Select the flavour you want!
